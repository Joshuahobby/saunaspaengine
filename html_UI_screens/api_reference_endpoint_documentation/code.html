Create a usage record

POST /v1/records

Creates a new usage record for a spa facility member. This endpoint is used to track entry, exit, and specific amenity usage (e.g., private sauna sessions, towel service) for billing and analytics.

key Authentication

This endpoint requires a Bearer Token associated with your API key. See Authentication Guide for more details.

Request Parameters

Field Type Description
member_id string REQUIRED The unique identifier of the member accessing the facility.
activity_type enum Type of activity. Options: entry, exit, service.
facility_id string The ID of the specific facility or branch.
timestamp ISO 8601 Optional. Defaults to current server time if not provided.

Possible Responses

200 OK The record was created successfully. expand_more
401 Unauthorized Invalid API key or missing authentication header. expand_more
404 Not Found The specified member_id does not exist. expand_more
terminal Code Samples 
curl --request POST \
  --url https://api.saunaspa.io/v1/records \
  --header 'Authorization: Bearer YOUR_TOKEN' \
