auto_mode Auto-dismiss Guidelines

  • 01

    Used for non-critical confirmations like "Saved", "Copied", or "Sent".

  • 02

    Standard duration is 3000ms (3 seconds) before fading out.

  • 03

    Pause timer on hover to allow users to read longer notification text.

priority_high Persistent Alert Rules

  • 01

    Mandatory for Errors that block the primary user workflow.

  • 02

    Must contain an explicit 'Close' or 'Dismiss' action button.

  • 03

    Place at the top of the content area or fixed to the viewport bottom.
